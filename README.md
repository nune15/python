# Matrix Inversion using Fadeev Method

This project implements matrix inversion using the Fadeev algorithm in Python.

## 📌 Description

The Fadeev method is an algorithm used to compute the inverse of a matrix through a recursive and determinant-based approach. It is especially useful for educational purposes and understanding linear algebra concepts.

This repository provides a simple and clear implementation of the algorithm.

## ⚙️ Features

- Matrix inversion using Fadeev method
- Pure Python implementation
- Easy to understand code
- Suitable for students and learning purposes

## 🧮 Algorithm

The algorithm works by:
1. Computing coefficients of the characteristic polynomial
2. Using recursive matrix operations
3. Deriving the inverse matrix from the results
